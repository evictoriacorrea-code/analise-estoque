# Projeto: Gestão Estratégica de Estoque 

## Objetivo
Este projeto tem como objetivo realizar o diagnóstico, análise e visualização de dados de estoque para mitigar riscos de ruptura e otimizar o capital de giro da operação. Através da identificação de gargalos de fornecimento e desbalanceamento de estoque, o projeto propõe ações táticas para melhoria do planejamento.

## Estrutura do Repositório

*   `AnaliseEstoqueSumig.ipynb`: Notebook Jupyter contendo a manipulação, tratamento e análise exploratória dos dados originais.
*   `dados.xlsx`: Base de dados bruta utilizada para o processamento - Fornecida pela sumig
*   `tabela_consolidada_dashboard.csv`: Dataset final tratado e preparado para consumo pelo Power BI.
*   `DashSumig.pbix`: Arquivo original do Power BI Desktop com o modelo de dados e visualizações interativas.
*   `DashSumig.pdf`: Exportação do dashboard em formato estático.
*   `Gestao-Estrategica-de-Estoque-SUMIG.pdf`: Apresentação executiva focada no diagnóstico e plano de ação.

## Tecnologias Utilizadas
*   **Python (Pandas/Jupyter):** Tratamento de dados e análise de métricas.
*   **Power BI:** Criação do dashboard de monitoramento e visualização de risco.
